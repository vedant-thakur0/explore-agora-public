# FEC Directory: Experimental Placeholder

**Status:** Experimental placeholder — not wired into the main AGORA pipeline.

This directory is reserved for future Federal Election Commission (FEC) data ingestion experiments. Nothing in the AGORA pipeline imports or depends on this directory.

**Intended scope:** Future experiments ingesting FEC contribution/expenditure data to complement the legislative dataset (e.g. linking campaign finance flows to AI-related bill sponsors). No roadmap or timeline exists yet.

Before building on it, open a GitHub issue to discuss with the project maintainer.
